# Disclaimer

This repository documents web application security testing performed in deliberately vulnerable training environments.

All activity described here was carried out against systems intended for security education and authorised testing. No production, customer, employer, university, or unrelated third-party systems were targeted.

The material is provided for:

- Education and professional development
- Defensive security understanding
- Authorised penetration testing
- Understanding common web application trust failures and mitigations

Do not use the techniques in this repository against systems, accounts, networks, or services without explicit authorisation.

Lab credentials, tokens, authorization codes, reset values, hostnames, and other artefacts shown in the reports belong only to temporary training environments. Sensitive or short-lived values are generally redacted because the technical behaviour, rather than the synthetic secret itself, is what matters.

The author assumes no responsibility for unauthorised or unlawful use of the material.
